About DiFri

DiFri is an open-source distraction-free browser built with Python, PyQt6, and QWebEngineView. It integrates DNS-level filtering and webapp isolation, designed to replace native browsers while promoting productivity.

The browser is lightweight, cross-platform (Windows, macOS, Linux), and easy to extend with additional PyQt6 widgets or custom styling. Perfect for educational institutions, families, or anyone seeking a cleaner browsing experience.

Initialized using uv for modern Python package management, DiFri opens the DiFri HomePage by default and includes intuitive navigation controls.
